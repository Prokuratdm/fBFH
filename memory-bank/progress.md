## Архитектура (после Итерации 12)
```
HomePage (shell с IndexedStack)
├── ClubsPage (для menuClubs/menuMyClub)
├── UsersPage (для menuUsers)
├── LocationsPage (для menuLocations, требует clubId)
├── InventoryPage (для menuInventory)
├── ExercisesPage (для menuExercises, split-view список+детали)
└── Placeholder «В разработке» (для остальных)
```

Меню (NavigationRail) всегда видно, навигация переключает индекс в IndexedStack.
Каждая страница имеет публичный `State.refresh()` и получает `GlobalKey` от HomePage — вызывается при `onDestinationSelected` для обновления данных.
