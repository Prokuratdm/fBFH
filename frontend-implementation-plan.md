# Frontend Implementation Plan: JBFH (Flutter)

План реализации фронтенда на Flutter с поддержкой Web + iOS + Android.  
Бэкенд: REST API (41 эндпоинт), JWT-аутентификация.

---

## Технологический стек

- **Flutter 3.x** (Dart)
- **State management**: Riverpod (рекомендуется) или Bloc
- **HTTP**: `dio` + `dio_cookie_manager`
- **Роутинг**: `go_router`
- **Drag-and-drop**: `super_drag_and_drop` (упражнения в тренировку)
- **Аутентификация**: хранение JWT в `flutter_secure_storage`
- **Платформы**: Web, iOS, Android

---

## Роли и доступные экраны

### ROLE_ADMIN (администратор системы)
Может всё. Без привязки к клубу.

| Экран | Функции |
|-------|---------|
| **Dashboard** | Общая статистика системы |
| **Clubs** | CRUD клубов, загрузка логотипов |
| **Users** | Создание/просмотр пользователей, смена пароля |
| **All Teams** | Просмотр команд всех клубов |
| **Inventory** | Управление общим инвентарём |
| **Exercises** | Управление упражнениями (все), загрузка картинок |
| **Standards** | CRUD нормативов |
| **Template Trainings** | CRUD эталонных тренировок, управление упражнениями |
| **Training Programs** | CRUD программ обучения |

### ROLE_METHODIST (методист)
Без клуба. Видит всё, но редактирует ограниченно.

| Экран | Функции |
|-------|---------|
| **Dashboard** | Общая статистика, программы обучения |
| **Clubs** | Просмотр клубов |
| **All Teams** | Просмотр команд всех клубов |
| **Exercises** | Просмотр всех упражнений, создание общих |
| **Standards** | CRUD нормативов |
| **Template Trainings** | CRUD эталонных тренировок |
| **Training Programs** | CRUD программ обучения |

### ROLE_CLUB (представитель клуба)
Привязан к клубу. Управляет клубом, командами, пользователями.

| Экран | Функции |
|-------|---------|
| **My Club** | Детали клуба, редактирование |
| **My Teams** | CRUD команд клуба, логотипы |
| **Team Coaches** | Назначение/удаление тренеров команды |
| **Team Children** | CRUD детей в команде |
| **Child Stats** | История роста/веса ребёнка |
| **Child Standards** | Результаты нормативов ребёнка |
| **Trainings** | CRUD тренировок команды (календарь) |
| **Training Builder** | Конструктор тренировки с drag-and-drop упражнений |
| **Locations** | Управление локациями клуба |
| **Users** | Создание пользователей клуба |
| **Inventory** | Управление инвентарём клуба |
| **Exercises** | Управление упражнениями клуба |
| **Standards** | Просмотр нормативов |
| **Template Trainings** | Просмотр и применение эталонных тренировок |
| **Training Programs** | Просмотр программ обучения |

│   └── widgets/                      # Переиспользуемые виджеты
├── features/

│   │   │   ├── stats/                # Рост/вес
│   │   │   └── standards/            # Результаты нормативов
│   │   ├── coaches/                  # Тренеры
│   │   ├── trainings/                # Тренировки (календарь)
│   │   │   └── builder/              # Конструктор тренировки
│   │   └── locations/                # Локации
│   ├── inventory/                    # Инвентарь
│   ├── exercises/                    # Упражнения
